# Cryptic transmission before detection (CTW framework)

This repository contains Python scripts to reproduce all main and
supplementary figures in the manuscript:

> Cryptic transmission before detection in host–vector–pathogen systems

## Contents

### Scripts
Each figure is generated by a single standalone script:

- `figure1_bw.py` – Fig. 1: Temporal decoupling example
- `figure2_bw.py` – Fig. 2: Parameter-space structure
- `figure3_bw.py` – Fig. 3: Management strategy comparison
- `figureS1_bw.py` – Fig. S1: Network robustness
- `figureS2_bw.py` – Fig. S2: Latency distribution robustness
- `figureS3_bw.py` – Fig. S3: Supplementary economic / sensitivity analyses
